Project #1: A data structure a day

Day #1:

So today's the first day of the big "let's make shit happen" week, starting with a simple linked list based stack implementation was a great warmup, let's see what I learned.
This article won't be as long as it's supposed to be, because the simplicity of the DS (data structure).

3 things every programmer should know about stacks:

  • Basic LIFO data structure, implemented in hardware in a great deal of processors
  • Every operation is O(1), no matter the case
  • 2 implementation exists: array-based and list-based (first has stack overflow issues)
I have nothing more to say, so let the code speak! On the project page you can access the proper repository.Check the source, I made the implementation so trivial that every C-based language programmer should understand it.
Tomorrow we will implement stack's complete opposite, the FIFO Queue, he is the polite all-knowing englishman of data structures, his power lies in it's simplicity.(A* path-finding, I'm slowly getting closer)

Things needed for implementing data structures in C++:

Knowledge of OOP and pointers, and a good deal of tolerancy for SEGFAULT's :D
